DOCTYPE statement
an "information" to the browser about what document type to expect. In HTML 5, it looks like this in the start of a document: <!DOCTYPE html>. Reference
<ol>
used for ordered lists by having a <ol> tag; and, within the opening and closing pair of the <ol>, you have list items that are each inside a <li> tag pair. The list items will each be shown with numbers in front and in the order that you list them in your code.
<ul>
used for unordered lists by having a <ul> tag; and, within the opening and closing pair of the <ul>, you have list items that are each inside a <li> tag pair. The list items will each be shown with bullets in front in the order you list them in your code.
<dl>
used for description lists by having a <dl> tag that identifies the list; and, within the opening and closing pair of the <dl>, you have terms that are each inside a <dt> tag pair and a description for the term in a <dd> tag pair after the <dt>tags. Reference
text editor
a type of computer program that edits plain text. Reference
word processor
a computer program that provides for input, editing, formatting, and output of text. Reference
client-server model
a distributed application structure that partitions tasks or workloads between the providers of a resource or a service, called servers, and service requesters, called clients. A server host runs one or more server programs, which share their resources with clients. A client usually does not share any of its resources, but it requests a content or service from a server. Reference
