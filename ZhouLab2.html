How to define a table?

<tr>
defines rows in a table
<th>
defines header cells in a table
<td>
defines normal cells in a table
<thead>
defines the header in a table
<tfoot>
defines the footer in a table
<caption>
provides a title or decription for the table

What is the diffeence between colspan and rowspan?

colspan
combines cells horizontally across columns
rowspan
combines cells vertically across rows

What is the difference between cells spacing and padding?

cells spacing
determines the space between cells in a table
padding
determines the space between content in a cell and the cell's border

How do you define table borders using CSS?

table {border-collapse: COLLAPSE OR SEPARATE; border-spacing: #; border: #px solid COLOR;}

How do you control border widths using CSS?

in the brackets with whatever selector you are trying to style, write "border-width" and input the desired size
